# Number Analysis

A real-time statistical analysis tool for numerical sequences from various regions.

## Features
- 🌓 Dark/Light mode toggle
- 📊 Real-time statistical analysis
- 🎨 Color-coded number visualization
- 📱 Fully responsive design
- 🔄 Auto-refresh every 5 minutes
- 📈 Statistical insights (sum, odd/even, high/low ratios)

## Local Development
1. Clone the repository
2. Open `index.html` in your browser
3. No build process required!

## Data Source
The application fetches data from a public API that provides numerical sequence results from multiple regions.
